# Client

### Required

Check that you have installed:

+   nodejs

### Run

For start client:

1. 	go to project directory
1.  run `$ npm install`
1.  run `$ node index -h` to get help
1.  run `$ node index` to start client

###

This app was tested under Ubuntu 14.04.3 LTS (trusty), Mac OS X 10.10.5 (Yosemite)
